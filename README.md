# Revisia 🎓

> Application mobile intelligente d'assistance à la révision universitaire.

---

## Le problème

Les étudiants reçoivent des dizaines de PDF de cours, révisent souvent tard,
et perdent un temps précieux à identifier l'essentiel.
Les outils IA existants (ChatGPT, NotebookLM) sont puissants mais pas conçus
pour la révision universitaire mobile et francophone.

---

## La solution

Revisia transforme automatiquement un cours PDF en matériel de révision exploitable :

- 📄 **Résumé structuré** du cours
- 🎯 **Points clés** identifiés automatiquement
- ❓ **Quiz / QCM** générés par IA
- 📱 **Expérience mobile-first**, simple et rapide

---

## Stack technique

| Couche | Technologie |

| Backend | Django REST Framework |
| IA | Groq API |
