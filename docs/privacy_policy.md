# Politique de confidentialité — Revisia

> Version 1.0 — Document en cours de rédaction

---

## 1. Données collectées

Revisia collecte uniquement les données nécessaires au fonctionnement :

- Prénom et adresse email (création de compte)
- Fichiers PDF uploadés (traitement temporaire uniquement)
- Historique des quiz et résumés générés

---

## 2. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

- Permettre l'accès au compte utilisateur
- Générer les résumés et quiz à partir des cours
- Améliorer l'expérience de révision

Aucune donnée n'est vendue ou transmise à des tiers.

---

## 3. Fichiers PDF

Les fichiers PDF uploadés sont :

- Traités temporairement pour extraction du texte
- Supprimés automatiquement après traitement
- Jamais stockés de manière permanente
- Jamais utilisés pour entraîner des modèles IA

---

## 4. Sécurité

Revisia applique les mesures suivantes :

- Mots de passe hashés (jamais stockés en clair)
- Communications chiffrées via HTTPS
- Clés API et secrets hors du code source
- Accès aux données strictement contrôlé

---

## 5. Droits de l'utilisateur

Chaque utilisateur peut à tout moment :

- Accéder à ses données personnelles
- Demander la suppression de son compte
- Demander la suppression de ses données
